# P1's cluster documentation

This repository contains the documentation for the P1's cluster. It is built using [Astro](https://astro.build/) with the [Starlight](https://starlight.astro.build/) documentation theme. The documentation is hosted on [GitHub Pages](https://pages.github.com/).

```bash
# Clone the repository
git clone git@github.com:Pioneer-Centre-for-AI/p1-compute-documentation.git

cd p1-compute-documentation

# Install dependencies
npm install

# Start development server
npm run dev
```

The documentation will be available at `http://localhost:4321`.
