Welcome to the P1's private documentation!
==========================================

The **Pioneer Centre for Artificial Intelligence** (P1 for short) is a Danish cross-university research centre. This private documentation is for all affiliates to share and communicate. Access is controleed through the GitHub repo and organization. See the :doc:`contributing` page to learn how to contribute to the documentation.

Unless you are actively working on documentation pages, you should prefer the stable branch. You can check and switch versions in the lower left corner.
